# ESP32 Firmware

This firmware handles:
- Sensor data acquisition (moisture + DHT11)
- Communication with Google Sheets via Apps Script
- Servo control based on remote commands

## Features
- WiFi-based cloud logging
- Real-time command polling
- Servo-based automation

## Hardware Used
- ESP32
- DHT11 Sensor
- Moisture Sensor
- Servo Motor

## Setup
1. Add WiFi credentials
2. Add Google Script Web App URL
3. Upload to ESP32
