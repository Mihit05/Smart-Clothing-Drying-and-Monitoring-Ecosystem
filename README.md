# Smart Clothing Drying & Monitoring System

## Overview
An IoT-based system that automates clothes drying using weather prediction and remote control.

## System Architecture

ESP32 → Google Apps Script → Google Sheets → ML Model → Command → ESP32

## Features
- Real-time sensor monitoring
- Cloud-based control system
- Rain prediction using ML
- Automated servo-based drying mechanism

---

## Components

### 1. ESP32 Firmware
- Reads sensors
- Sends data to cloud
- Executes commands

### 2. Cloud Backend
- Google Apps Script API
- Data storage in Sheets

### 3. Machine Learning Model
- Predicts rainfall
- Helps automate decisions
